## 📝 TypeScript
- TypeScript는 JavaScript의 슈퍼 셋이다. 이외에 추가적인 기능 제공
- 타입을 제공 → 명시적인 데이터에 대한 유형 설명
  - 안정성 있는 코드 작성 가능
  - 예상치 못한 오류 발생을 줄일 수 있다.
- 컴파일 언어
- 객체 지향 프로그래밍 언어
